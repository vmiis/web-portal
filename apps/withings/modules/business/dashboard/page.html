Sleep API Reference

Provides the sleep measures for the night of the user, with the details of each phase of their sleep cycle. Note: if startdate and enddate are separated by more than 24h, enddate will be overwritten with 24h past startdate.

Start Date: Measures' start date.
End Date: Measures' end date.
Data Fields: List of requested data fields, separated by a comma.

Sleep summary API Reference

Get the basic information about a night of the user, such as the total time they slept, how long it took them to fall asleep, etc.

Start Date: Start date.
End Date: End date.
Data Fields: List of requested data fields, separated by a comma.
