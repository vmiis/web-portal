This micro web application contains the following files

Clinet Side
  • index.html layout for header, contents and footer
  • header.htm logo, menu, login/logout etc
  • footer.htm copyright info, links etc
  • home.htm landing page, show some information
  • data.html view and search data in a cloud database table
  • form.html data collection form
Server Side
  • server.js providing web hosting services
  • start-server.bat start server
Tools
  • editor.html content source file editor


Download and run this micro web app on a local computer
