  • Create, modify, view and submit requests.
    Users can only see the records created by themself.
    Need to select a manager for approving and an processor for processing.
    The requests can only be viewed once approved.
  • A notification email will be send to the manager.
  • Approving request.
    A manager can only see the records that need to be approved by him/her.
    The requests can only be viewed once processed.
  • A notification email will be send to the processor.
  • Processing requests.
    A processor can only see the records that need to be processed by him/her.
  • A notification email will be send to the requester once the request has been processed.
