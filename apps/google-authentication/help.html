How to sign in with Google authentication

  1. Create authorization credentials

  2. Add google api js library in index.html 
        <script async defer src="https://apis.google.com/js/api.js"
            onload="this.onload=function(){};handleClientLoad()"
            onreadystatechange="if (this.readyState === 'complete') this.onload()">
    </script>
  3. Add your app infomation in the google.js 
        'apiKey': 'YOUR_API_KEY',
    'clientId': 'YOUR_CLIENT_ID',
  4. Once you have signed in successfully you can start to use api to exchange data with Google endpoint
    The following is an example to get the information about yourself 
        $vm.load_module('json_viewer','',{json:GoogleAuth.currentUser.get().getBasicProfile()})
